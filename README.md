# personal_website

我的个人网站 — terminal 风格 layout + 文字粒子 avatar。

## 本地预览

```bash
npx serve .
```

然后打开 http://localhost:3000。

## 文件结构

```
.
├── index.html              ← 入口页面
├── styles/
│   └── main.css            ← 全部样式
├── scripts/
│   ├── avatar.js           ← canvas 文字粒子 avatar
│   └── boot.js             ← terminal 启动序列动画
└── README.md
```
