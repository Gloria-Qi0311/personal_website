# personal_website

My personal website — terminal-style layout with a text-particle avatar.

## Local preview

```bash
npx serve .
```

Then open http://localhost:3000.

## File structure

```
.
├── index.html              ← entry page
├── styles/
│   └── main.css            ← all styles
├── scripts/
│   ├── avatar.js           ← canvas text-particle avatar
│   └── boot.js             ← terminal boot-sequence animation
└── README.md
```
